📊 Student Performance Dashboard – Tableau Project

The Student Performance Dashboard is an interactive data visualization project created using Tableau Public to analyze and monitor student academic performance through multiple educational factors. The dashboard transforms raw student data into meaningful visual insights that help understand trends, comparisons, and overall performance patterns effectively.
This project includes various charts and graphs such as stacked bar charts, grade distribution analysis, attendance versus final score comparison, and department-wise performance visualization. The dashboard allows users to interact with filters like gender and grade categories, making the analysis more dynamic and user-friendly.
One of the main objectives of this project is to study the relationship between attendance and final academic scores. The dashboard also compares student performance across different departments including Computer Science, Civil Engineering, Electrical Engineering, and Mechanical Engineering. In addition, it analyzes how parental education level may influence student academic outcomes.
The project demonstrates important data analytics and dashboard design skills by combining clean visual layouts, interactive elements, and effective storytelling through data. It provides a professional and visually appealing interface that can help educational institutions, teachers, and administrators make better academic decisions based on data insights.

🔹 Key Features

* Interactive dashboard with filters
* Attendance vs Final Score analysis
* Department-wise performance comparison
* Student grade distribution visualization
* Gender-based academic analysis
* Parent education impact analysis

🛠️ Technologies Used

* Tableau Public

This project showcases practical knowledge of Tableau, data visualization techniques, and analytical reporting while presenting educational data in an engaging and understandable format.
